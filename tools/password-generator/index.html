Générateur de Mots de Passe

Sécurité du mot de passe

Ce générateur utilise l'API Web Cryptography de votre navigateur pour produire des mots de passe cryptographiquement sûrs. Contrairement à la commande OpenSSL dans un terminal Linux, les mots de passe sont générés directement dans votre navigateur sans être envoyés sur un serveur. Cela garantit que vos mots de passe ne quittent jamais votre appareil, mais la qualité de l'aléatoire dépend des capacités de votre navigateur et système.

Générateur Hexadécimal

Génère un mot de passe de 32 octets encodés en hexadécimal

Détails du mot de passe Hexadécimal :

  • Longueur : 64 caractères (32 octets encodés en hexadécimal)
  • Caractères : Chiffres de 0 à 9 et lettres de a à f
  • Avantages : Format standard pour les clés cryptographiques, facilement lisible
  • Inconvénients : Moins compact que d'autres encodages
  • Limites : Moins de variété de caractères que d'autres formats

Générateur Base64

Génère un mot de passe de 32 octets encodés en Base64

Détails du mot de passe Base64 :

  • Longueur : Environ 44 caractères (32 octets encodés en Base64)
  • Caractères : Lettres majuscules et minuscules, chiffres, et les symboles + et /
  • Avantages : Format couramment utilisé dans les applications web et les clés API
  • Inconvénients : Moins dense que l'hexadécimal pour certains usages
  • Limites : Le caractère + peut poser problème dans certaines URL si non encodé
